# client

## About
This DIR contains the Twitch panel extension built using vueJS<br>
>

## Project setup
```
npm install
```
<br>
Then load the project within vue-cli ui

```
vue ui
```

Run the serve task with the following options:
- host: localhost
- port: 8090
- https activated
